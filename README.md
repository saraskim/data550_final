# Leveraging wastewater data to predict norovirus cases

> Evaluates and compares model performance for predicting norovirus cases one week ahead using case data, wastewater data, or both

------------------------------------------------------------------------

## Initial code description

`code/1_setup.R`

  - Reads in raw data `data/noro_for_data550.csv`
  - Cleans data and makes variables necessary for analysis
  - Saves cleaned dataset as a `.rds` object in `data/` folder
  
`code/2_split_data.R`

  - Reads in cleaned data previously produced in `code/1_setup.R`
  - Splits data into a training set and test set 
  - Saves cleaned datasets as a `.rds` objects in `data/` folder
  
`code/3_analysis.R`

  - Reads in training and test datasets previously produced in `code/2_split_data.R`
  - Includes three models: 1: prior cases as predictors, 2: prior norovirus levels in wastewater as predictors, 3: both
  - Runs defined models on training dataset
  - Predicts norovirus cases on test dataset
  - Saves predictions as `test_data_with_predictions.rds` objects in `data/` folder

`code/4_render.R`

  - Renders `report.Rmd`

`report.Rmd`

  - Reads in cleaned full dataset and model predictions generated by `code/1_setup.R` and `code/3_analysis.R`, respectively
  - Makes a time series figure for the observed case and wastewater data
  - Evaluates each model using the root mean squared error and outputs the results as a table
  - Makes a figure showing the model predictions against the observed data
